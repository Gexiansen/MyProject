# 家庭财务

这是一个单文件家庭财务管理工具，用来帮助家庭每月回答三个问题：

1. 我们现在处于什么位置。
2. 这个月为什么会发生这些变化。
3. 距离家庭目标和想要的生活还有多远。

应用入口位于 [`docs/index.html`](docs/index.html)，使用 React、Recharts、Tailwind CSS 和 Babel 的 CDN 版本，不需要安装依赖或执行构建。

## 开始使用

### 打开页面

可以直接使用浏览器打开 `docs/index.html`，也可以在项目目录启动本地静态服务：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://localhost:8000/docs/
```

页面依赖 CDN 加载运行库，首次打开时需要能够访问互联网。

### 解锁页面

- 默认密码：`123456`。
- 密码固定为 6 位数字，可以在桌面端顶部或手机端「更多」中修改。
- 刷新页面后需要重新输入密码。
- 该功能只是本机页面锁，不会加密财务数据，也不是服务端账户认证。

### 首次配置

没有本地数据时，页面会载入一套演示数据。正式使用前建议按以下顺序操作：

1. 打开「重置」，选择「清空全部，从零开始」。
2. 在「账户管理」中建立家庭的资产、负债、收入和支出账户。
3. 在「月度结账」中创建第一个月份并录入实际数据。
4. 根据需要在「家庭目标」中建立应急资金、教育、旅行或其他目标。
5. 完成首次数据导出，并把 JSON 备份到其他位置。

## 推荐的每月使用流程

页面顶部会显示最新财务数据月份及其状态，例如「数据截至 2026年4月 · 已结账」。这里表示全局最新数据的时间边界，不会随月度结账中选择的历史月份变化；没有月度记录时显示「尚未建立月度记录」。

### 1. 选择或创建月份

进入「月度结账」后，可以选择已有月份，也可以通过月份输入框创建新月份。

- 选择已有月份不受结账状态限制。
- 新建月份前，目标月份之前最近一个已有月份必须已经完成结账；未结账时「选择／新建」按钮会禁用并提示需要先完成的月份。
- 创建第一条月度记录，或补录早于当前最早记录的月份时，没有前置结账要求。

创建月份时：

- 资产和负债会沿用此前最近月份的余额，并标记为本月待核对。
- 固定支出会使用账户管理中设置的默认金额。
- 收入和非固定支出从 `0` 开始填写，并标记为本月待处理。

### 2. 按家庭成员录入

录入顺序按「老公」「老婆」「共同」分组，每组内部依次显示：

- 现金类资产：填写可随时用于生活或应急的月末余额。
- 投资资产：填写股票、基金等会随市场波动的月末余额。
- 固定资产：填写房产等通常无法快速变现的月末余额。
- 负债：填写月末余额。
- 收入：填写当月发生额。
- 支出：填写当月发生额。

手机端通过成员切换按钮一次显示一个成员，桌面端会连续展示全部成员。成员名称旁的橙色数字表示该成员仍有多少项系统初始数值待处理，全部处理后数字自动消失。

本月尚未重新输入的资产、负债、收入和非固定支出使用浅橙色输入框：资产、负债表示仍沿用上月余额，收入和非固定支出表示仍使用初始值 `0`。固定支出使用账户默认金额，不进入待处理统计。点击金额输入框时会自动选中原数字，直接输入即可整体覆盖，不需要先逐位删除。只有实际重新输入后才视为已经处理；仅点击输入框不会改变状态。每次修改都会自动保存，不需要手动点击保存。

录入区上方会汇总全家庭的现金类、投资和固定资产总额，并显示各大类较上月的变化；每个账户下方仍保留自身的上月余额和变化值。

修改最新月份的固定支出时，会同步更新该账户以后月份使用的默认金额；修改历史月份只影响当月。

### 3. 检查本月变化

录入完成后，页面会自动展示：

- 本月收入、支出和结余。
- 净资产变化。
- 结余与非收支变化的拆解。
- 对净资产影响最大的资产或负债账户。

可以为主要变化选择「市场涨跌」「主动转入／转出」「估值调整」「偿还／新增负债」「数据修正」或其他原因。

### 4. 完成本月结账

点击「完成本月结账」后，系统会检查所有启用账户是否已经填写，并提示以下异常情况：

- 启用账户存在空值。
- 资产或负债仍沿用上月余额，或者收入、非固定支出仍使用初始值 `0`。
- 本月收入为 `0`。
- 资产或负债余额为 `0`。
- 资产或负债较上月发生大幅变化。

空值属于阻断项，补齐后才能结账。系统会分别列出仍沿用上月的余额，以及仍使用初始值 `0` 的收入和非固定支出；可以返回修改，也可以一次点击「确认以上数值并结账」。确认后这些项目会记录为本月已经处理。其他提示确认符合实际情况后仍可结账。

完成后该月份会被锁定，避免误改。需要修正时可以点击「重新编辑」，修正后再次完成结账。

### 5. 完成月度复盘

结账后会出现：

- 月度家庭简报：总结当前位置、变化原因和下月行动，可以复制分享。
- 上月行动复盘：勾选上月行动是否完成，并查看关联指标的同期结果。
- 下月行动：最多记录 2 项，可以选择是否关联月结余、现金或负债指标。

关联指标只表示同期变化，不代表该行动直接导致了结果。

## 功能说明

### 总览

总览默认展示家庭最需要关注的信息：

- 净资产、本月结余、总资产和总负债。
- 总资产中的现金、投资和固定资产构成。
- 家庭目标所需资金与近期平均结余的关系。
- 本月财务关注，例如负结余、净资产下滑、现金缓冲不足、负债增加或投资过度集中；设置应急资金目标后，现金缓冲会使用家庭设定的保障月数判断。
- 数据核对，例如账户缺失、大额变化未解释、连续三个月余额不变、历史月份重新编辑或长期未备份。
- 财务自由进度和预计所需时间。

点击「展开深度分析」可以查看：

- 固定支出覆盖预估。
- 资产占比和资产结构变化。
- 净资产与负债趋势。
- 收入、支出、结余和储蓄率趋势。
- 净资产增长来源。
- 行动复盘。
- 成员账户归属摘要。
- 财务自由情景推演。

图表范围可以切换为「近一年」或「全部」。

### 账户管理

账户分为四类：

| 账户类型 | 填写口径 | 补充设置 |
| --- | --- | --- |
| 资产 | 月末余额 | 现金类、投资或固定资产 |
| 负债 | 月末余额 | 无 |
| 收入 | 当月发生额 | 无 |
| 支出 | 当月发生额 | 可设为固定支出并填写默认金额 |

每个账户必须选择「老公」「老婆」或「共同」，成员名称不会写进账户标题。

- 现金类资产：可以随时用于生活和应急，可作为应急资金来源。
- 投资资产：价格可能波动，不计入应急资金，可作为普通家庭目标的资金来源。
- 固定资产：通常无法快速变现，不作为家庭目标资金来源。

- 停用账户：保留全部历史数据，不再出现在后续月度录入中，可以重新启用。
- 删除账户：会同时删除该账户在所有月份中的历史数据，并改变历史统计结果。

如果只是暂时不再使用某个账户，优先选择停用。

### 家庭目标

家庭目标支持：

- 普通家庭目标和应急资金两种类型；旧数据中的目标默认按普通目标处理。
- 目标名称、目标金额和目标日期。
- 优先、正常、以后三级优先级。
- 手动填写当前已准备金额。
- 关联一个或多个符合口径的资产账户，自动汇总最新余额。
- 查看完成进度、资金缺口和按期每月需准备金额。
- 查看每次结账保存的目标快照以及实际进度与计划的差异。

应急资金可以设置保障月数，系统按最近最多 12 个月平均月支出计算建议金额，并显示当前资金可覆盖的月份数。应急资金只能关联现金类资产；股票、基金和固定资产不会计入应急缓冲。普通目标可以关联现金或投资资产，固定资产不作为目标资金来源。

同一个资产账户默认只能关联一个目标，避免重复计算。总览中的「目标每月需准备」只汇总优先和正常目标，并按这些目标同时推进计算；「以后」目标不计入当前资金压力。若旧数据或导入文件中存在不符合新口径的关联账户，资金不会计入目标进度，总览会提示重新选择来源，但不会删除原始目标数据。

### 年度小结

年度小结按年份汇总：

- 每个统计年度的实际覆盖月份；完整年度显示全年，不完整年度显示起止月份和已记录月数。
- 期末净资产及较上年末变化。
- 年度收入、支出和结余。
- 净资产增长中自己攒下的部分与非收支变化部分。
- 年度储蓄率。
- 期末总资产、负债、区间减债和投资集中度。

年度小结中的「较上年末」是本统计期末净资产与上一年度最后一条记录的比较，不是同期月份同比；不完整年度的收入、支出、结余和储蓄率只汇总页面标明的实际月份区间。

手机端的年度明细表可以左右滑动查看。

## 主要计算口径

```text
净资产 = 总资产 - 总负债
本月结余 = 本月收入 - 本月支出
净资产变化 = 本月净资产 - 上月净资产
非收支变化 = 净资产变化 - 本月结余
月度储蓄率 = 本月结余 ÷ 本月收入
```

当月收入为 `0` 时，不计算该月储蓄率。

```text
应急资金建议金额 = 最近最多 12 个月平均月支出 × 保障月数
应急资金覆盖月数 = 当前已准备的现金资金 ÷ 最近最多 12 个月平均月支出
```

没有历史支出或平均月支出为 `0` 时，不计算覆盖月数，也不会自动生成大于 `0` 的建议金额。

「非收支变化」可能包含投资涨跌、房产估值、负债调整、账户增删或余额修正，不能直接理解为投资收益。

投资集中度按照「最大单一投资账户余额 ÷ 投资资产总额」计算，超过 `50%` 时只做关注提示，不代表统一投资标准。

### 财务自由计算

```text
年支出 = 最近最多 12 个月平均月支出 × 12
财务自由目标本金 = 年支出 ÷ 提取率
预计年被动收入 = 当前投资资产 × 提取率
```

提取率可以在 `3%` 到 `6%` 之间调整。预计所需时间按近期平均结余线性推算，不计未来投资收益和支出变化。

保守、基准、乐观情景用于观察不同收益率和支出增长率假设的影响，未计税费、投资波动和家庭计划变化，不是收益承诺。

「固定支出覆盖预估」使用最近 3 个月平均收入减去当前固定支出，只衡量固定支出覆盖能力，不包含日常开销等浮动支出。

## 数据保存与备份

### 自动保存

财务数据自动保存在当前浏览器的 `localStorage` 中。以下情况可能导致页面读取不到原数据：

- 更换浏览器或设备。
- 清除浏览器站点数据。
- 更换页面地址、端口或打开方式。
- 使用无痕模式后关闭窗口。

因此，浏览器本地数据不能替代备份。

### 导出

桌面端使用顶部「导出」，手机端使用「更多 → 导出数据」。导出的文件名类似：

```text
family-finance-2026-07-16.json
```

JSON 包含账户、月度记录、结账状态、余额核对状态、家庭目标类型与保障月数、行动、变化原因、目标快照、提取率和最近导出时间。页面密码不会写入导出文件。

建议至少每月结账后导出一次，并把备份保存在另一个目录或设备中。

### 导入

导入有效的家庭财务 JSON 后，页面会用文件中的账户、记录和设置替换当前状态。导入前应先导出当前数据，避免覆盖后无法恢复。

导入不会修改当前浏览器中保存的页面密码。

### 重置

重置提供两个选项：

- 恢复为演示数据。
- 清空全部，从零开始。

这两个操作都会覆盖当前财务数据，但不会重置页面密码。执行前应先导出备份。

## 使用边界

- 这是家庭财务记录和复盘工具，不提供投资建议、收益保证或财务顾问服务。
- 财务自由结果和情景推演都是基于当前数据的估算，不是未来承诺。
- 成员归属摘要只反映账户分类，不代表夫妻个人贡献或实际分摊。
- 页面密码只是本机访问门槛，不是数据加密。
- 财务 JSON 包含真实家庭财务信息，请勿上传到公开仓库或发送给无关人员。
