# 家庭财务项目 Roadmap

## 当前阶段

单文件家庭财务应用持续迭代阶段，重点优化手机端月度录入和家庭财务分析体验。

## 已完成

- 单文件 React 家庭财务应用，支持总览、账户管理、月度结账和年度小结。
- 账户与月度数据自动保存到浏览器 `localStorage`。
- 财务数据 JSON 导入、导出、恢复演示数据和清空重建。
- 六位数字本机页面锁及修改密码功能。
- 月度结账支持选择任意月份、自动沿用余额和固定支出默认值。
- 固定支出默认金额与最新月份双向同步，历史月份修改不影响默认金额。
- 月份删除二次确认。
- 总览包含净资产、资产结构、现金流、增长来源、储蓄率和持仓集中度提醒。
- 总览图表支持近一年与全部范围，并适配手机宽度。
- 月度结账按老公、老婆、共同分区，成员内部按资产、负债、收入和支出排列。
- 资产和负债显示上月余额与本月变化，首月显示无上月数据提示。
- 月度结账支持“录入中／已结账”状态，状态随本机数据自动保存并纳入 JSON 导入导出。
- 结账前检查启用账户完整性，并对零收入、零余额和资产负债大幅变化给出可确认提醒。
- 已结账月份默认锁定录入，支持“重新编辑”；旧数据和旧版 JSON 默认保持“录入中”。
- 净资产增长来源统一使用“非收支变化”，避免将估值、账户调整等误称为市场收益。
- 总览和月度结账自动展示净资产变化公式及影响最大的资产、负债账户。
- 财务自由进度补充目标本金差距、被动收入缺口、年度支出覆盖月数和预计所需时间。
- 财务自由预计时间沿用当前线性储蓄口径，并明确不计未来投资收益和支出变化。
- 总览根据最新结账数据生成“本月财务关注”，集中呈现需要优先核对的现金流、资产和负债变化。
- 财务关注规则直接展示触发原因和阈值，不新增评分、手工录入或持久化字段。
- 总览“更多分析”按老公、老婆、共同展示最新月份净资产、负债、收入、支出和归属收支差。
- 成员摘要完全沿用账户 `member` 字段，共同支出不虚构夫妻分摊或个人贡献。
- 家庭目标支持新增、编辑、删除确认、优先级、完成进度和按目标日期倒推月投入。
- 家庭目标及其进度自动保存并纳入 JSON 导入导出，旧版数据无需迁移。
- 月度结账支持最多两项下月行动，并在下个月结账时复盘完成状态。
- 主要资产、负债变化支持可选原因标注，并同步展示在总览归因和图表提示中。
- 财务自由增加保守、基准、乐观三种复利与支出增长情景，并明确展示假设边界。
- 总览将家庭目标所需月投入与最近 12 个月平均结余比较，直接显示当前资金余量、缺口或已到期状态。
- 月度结账自动生成“当前位置、变化原因、下月行动”三段式家庭简报，并支持复制和失败提示。
- 完成月度结账时保存家庭目标快照，展示实际增加、计划增加、执行状态和最近进度记录。
- 家庭目标可关联资产资金账户并自动汇总余额，默认禁止同一账户重复归属多个目标。
- 总览提供数据可信度事实检查，覆盖缺失值、未解释变化、余额不变、重新编辑、目标关联和备份时间。
- 行动项支持选择观察指标，并汇总完成率、同期指标改善和逾期未完成事项。
- 总览默认聚焦本月结论与目标距离，图表、成员摘要和行动复盘统一收纳到深度分析。
- 月度结账改为“选择月份—按成员录入—变化解释—完成结账—简报与行动”的操作顺序，手机端支持老公、老婆、共同分段切换。
- 月度家庭简报、上月行动复盘和下月行动在完成结账后展示，历史月份默认折叠，减少录入前滚动距离。
- 手机端主导航精简为总览、月度结账、家庭目标和更多，账户管理、年度小结、导入导出、密码及重置收纳到更多菜单。
- 总览核心指标精简为净资产、月结余、总资产和总负债；现金流覆盖预估、复利情景统一收纳到深度分析。
- 财务自由预计结果改为时间估算，情景推演只展示约需年数，不再显示缺乏确定性的具体月份。
- “下月现金流预测”改为“固定支出覆盖预估”，明确结果不包含浮动支出。
- 行动分析统一改为“行动复盘”，只展示完成情况和同期指标结果，不再统计或暗示行动有效性。
- 月度储蓄率折线已合并到收入、支出与结余趋势图，使用独立百分比坐标轴和完整 Tooltip；独立储蓄率图与重复的集中度趋势图已移除。
- 数据核对仅在异常时展开详情，余额不变规则改为连续三个月且排除固定资产。
- 家庭目标月投入只汇总优先和正常目标，并明确按同时推进计算；已有目标时新增表单默认收起。
- 账户管理移除与总览重复的家庭总体状态，用户进入页面后直接管理账户。
- 新增面向用户的 `README.md` 使用说明，覆盖首次配置、月度结账流程、功能入口、计算口径、数据备份和使用边界，并纳入项目同步维护规范。
- 新建月份增加前置结账依赖：目标月份之前最近一个已有月份必须已结账；已有月份仍可直接选择，首次记录和更早历史补录不受限制。
- 资产口径进一步拆分为现金类、投资和固定资产；账户编辑时明确流动性及目标资金用途，旧账户缺少大类时按现金类兼容读取。
- 月度结账在每个成员下按现金类、投资、固定资产、负债、收入和支出分层录入，并新增全家庭三类资产总额及环比变化汇总。
- 家庭目标增加普通目标和应急资金类型；应急资金仅允许现金来源，普通目标排除固定资产，不符合口径的旧关联保留但不计入进度并提示修正。
- 应急资金支持保障月数、按最近最多 12 个月平均支出计算建议金额和当前覆盖月数，并同步展示在家庭目标与总览。
- 总览的现金缓冲提醒在存在应急资金目标时使用家庭保障月数和实际准备资金，否则沿用默认 3 个月提示线。
- 新建月份会记录资产、负债、收入和非固定支出的系统初始状态；待处理输入框统一使用橙色样式，成员标签显示待处理数量，重新输入后实时减少。
- 月度金额输入支持聚焦后直接覆盖原数字；仅点击不视为处理，结账时分别列出沿用余额和仍为初始值 `0` 的收入、非固定支出并可一次确认，固定支出不参与统计。
- 顶部标题显示全局最新数据月份及结账状态；年度小结显示各年度实际覆盖月份，并统一使用期末、区间和较上年末口径，避免将不完整年度误解为全年数据。
