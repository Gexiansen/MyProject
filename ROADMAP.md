# 家庭财务项目 Roadmap

## 当前阶段

单文件家庭财务应用持续迭代阶段，重点优化手机端月度录入和家庭财务分析体验。

## 已完成

- 单文件 React 家庭财务应用，支持总览、账户管理、月度结账和年度小结。
- 账户与月度数据自动保存到浏览器 `localStorage`。
- 财务数据 JSON 导入、导出、恢复演示数据和清空重建。
- 六位数字本机页面锁及修改密码功能。
- 月度结账支持选择任意月份、自动沿用余额和固定支出默认值。
- 固定支出默认金额与最新月份双向同步，历史月份修改不影响默认金额。
- 月份删除二次确认。
- 总览包含净资产、资产结构、现金流、增长来源、储蓄率和持仓集中度分析。
- 总览图表支持近一年与全部范围，并适配手机宽度。
- 月度结账按老公、老婆、共同分区，成员内部按资产、负债、收入和支出排列。
- 资产和负债显示上月余额与本月变化，首月显示无上月数据提示。
- 月度结账支持“录入中／已结账”状态，状态随本机数据自动保存并纳入 JSON 导入导出。
- 结账前检查启用账户完整性，并对零收入、零余额和资产负债大幅变化给出可确认提醒。
- 已结账月份默认锁定录入，支持“重新编辑”；旧数据和旧版 JSON 默认保持“录入中”。
- 净资产增长来源统一使用“非收支变化”，避免将估值、账户调整等误称为市场收益。
- 总览和月度结账自动展示净资产变化公式及影响最大的资产、负债账户。
- 财务自由进度补充目标本金差距、被动收入缺口、年度支出覆盖月数和预计达成月份。
- 财务自由预计月份沿用当前线性储蓄口径，并明确不计未来投资收益和支出变化。
